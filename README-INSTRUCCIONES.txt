INSTRUCCIONES PARA SUBIR TURISMO TRANSCELL A HOSTGATOR
============================================

Este archivo ZIP contiene la versión actualizada del sitio web "Turismo Transcell",
con la nueva funcionalidad de animación de avión al hacer clic en el botón COMPRAR AHORA.

CAMBIOS IMPLEMENTADOS:
- Ahora cuando se hace clic en el botón "COMPRAR AHORA", aparece una animación de un avión girando
- Se muestra el mensaje "Cargando las mejores ofertas, por favor espere..."
- La animación dura 5 segundos antes de redirigir a la página de destino

PASOS PARA SUBIR EL SITIO:

1. Inicia sesión en tu panel de control de Hostgator (cPanel)

2. Busca y haz clic en "Administrador de archivos"

3. Navega hasta la carpeta pública de tu sitio (normalmente '/public_html/' o la carpeta específica
   donde quieres alojar este sitio)

4. Sube el archivo ZIP (hostgator_final.zip)

5. Una vez subido, selecciona el archivo y haz clic en la opción "Extraer"

6. Los archivos se extraerán directamente en la carpeta actual. Asegúrate de que los archivos
   estén en la carpeta raíz y no dentro de una subcarpeta

7. Verifica que el archivo 'index.html' esté presente en la raíz de la carpeta

ESTRUCTURA DE ARCHIVOS:
- index.html (archivo principal)
- assets/ (contiene CSS y JavaScript)
- images/ (contiene imágenes e iconos)
- _redirects (archivo de configuración para redirecciones)

Si tienes algún problema o pregunta, por favor contacta al desarrollador.
